# 🎬 RENOX ENGINE

AI Video Editing Engine for Telegram.  
Sends back a 9:16 · 15s · 30fps short optimized for TikTok, Reels, and Shorts.

---

## Features

- **Blurred background** for horizontal videos — subjects preserved, never center-cropped
- **Auto-trim** — starts at 30% in to skip intros; fills 15 seconds
- **Speed ramp** — intelligently speeds up or slows down to fit 15s
- **4 styles** — sigma, cinematic, emotional, anime
- **80MB / 3-minute** input limit
- **Zero system dependencies** — `ffmpeg-static` bundles FFmpeg inside `node_modules`
- **Render Free** compatible — keep-alive ping, `veryfast` preset, ephemeral file handling
